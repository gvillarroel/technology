# Dual format standards

Recommended rule:

1. Author internal links once using the canonical HTML route.
2. Render the HTML page from the Markdown source.
3. Expose a sibling `.md` endpoint that rewrites internal links to `.md`.

Backlinks:

- [Return to overview](./overview)

This keeps authoring simple and avoids maintaining two independent navigation systems.
